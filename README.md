# Google Webmaster Tools Bulk Add Parameters Chrome Extension

## Installation
1. Download the extension from [github](https://github.com/montes/google-webmaster-tools-bulk-add-parameters/archive/master.zip) and unzip it.
2. Go to **chrome://extensions/** and turn on Developer mode.
3. Click on **Load unpacked extension . . .** and load the extension.

## Usage
1. Create a text file with a parameter on each line.
2. Go to google webmaster tools parameters add section.
3. Set options.
4. Click on Choose File button.
7. Select the file.
